# Contributing to AtlasForge Robotics

Thanks for considering a contribution. This project targets both real robots and simulators, so please keep safety, determinism, and reproducibility in mind.

## Attribution
If you publish or demo work based on this repository, please reference:
`https://github.com/MustafaAlpay/ai-robotics`

## Development Workflow
- Create focused branches and keep changes scoped to a single feature or fix.
- Run linters/tests before opening a PR (see `AGENTS.md` for commands).

## Commit Messages
- Use imperative, present-tense subjects (for example: `Add lidar cache`).
- Include a short body for behavior changes, hardware updates, or new interfaces.

## Pull Requests
- Describe the intent, design tradeoffs, and any affected platforms.
- Link issues or discussions where possible.
- Include test evidence (command output, logs, or screenshots for UI/sim changes).
- Highlight safety implications for robot control paths.

## Code of Conduct
Be respectful, constructive, and safety-minded in discussions.
